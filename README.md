# 💍 Wedding SaaS Platform

A modern SaaS-based Wedding Management Platform built using **Spring Boot**, **PostgreSQL**, and cloud image storage support. Users can create accounts, log in securely, create wedding events, and upload/manage wedding images in a scalable environment.

---

# 🚀 Features

## 🔐 Authentication & Authorization
- User Registration & Login
- Secure Authentication
- Session/JWT Based Authorization
- Protected APIs

## 💒 Wedding Management
- Create Wedding Events
- Update Wedding Details
- Delete Wedding Events
- View All Weddings
- Manage Wedding Information

## 🖼️ Image Upload System
- Upload Wedding Images
- Store Multiple Images Per Wedding
- Image Management APIs
- Cloud Storage Support
- Optimized Media Handling

## 📦 SaaS Architecture
- Multi-user Support
- Scalable Backend Architecture
- RESTful API Structure
- Modular Service Layer
